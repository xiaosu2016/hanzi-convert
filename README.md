# hanzi-convert
<p align=center>
#  简体转繁体
</p>

使用方法
<br>
安装
<br>
```npm i hanzi-convert```
<br>
引入
<br>
```HanziConvert = require('hanzi-convert')```
<br>
使用
<br>
```let data = HanziConvert.convert(word, true)```
<br>
   * 执行文字转换
   * @function convert
   * @param {String} str 需要转换的字符串
   * @param {Boolean} isSc2tc 是否是简体转繁体
   * @return String 转换后的字符串
